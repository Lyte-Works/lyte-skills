# CHRO Advisor

## When to Use
- Building an HR strategy for a growing company
- Hiring plans and job description writing
- Performance review framework design
- Culture and values documentation
- Organizational structure decisions

## Workflow

### Step 1: Organizational Assessment
- Current team size and structure
- Key roles filled and unfilled
- Cultural strengths and challenges
- Growth plan (how many hires in next 6-12 months?)

### Step 2: Hiring Plan
| Role | Priority | Timeline | Type | Status |
|------|----------|----------|------|--------|
| [Role] | High/Med/Low | Q1/Q2/Q3 | FT/PT/Contract | Open/Filled |

For each role, define:
- **Job description**: responsibilities, requirements, nice-to-haves
- **Compensation range**: research market rates
- **Interview process**: stages, questions, evaluation criteria
- **Onboarding plan**: first week, first month, first quarter

### Step 3: Performance Framework
Simple, actionable performance management:
- **Quarterly goals**: 3-5 measurable goals per person
- **Monthly 1:1s**: progress check, blockers, feedback
- **Quarterly reviews**: goal assessment, development feedback
- **Annual compensation review**: based on performance and market data

### Step 4: Culture Documentation
Document in Notion:
- Company values (from Company OS skill)
- How values translate to behavior
- Communication norms (meetings, async, tools)
- Decision-making process
- Conflict resolution approach

### Step 5: People Operations
| Process | Documentation |
|---------|-------------|
| Onboarding checklist | First day, week, month tasks |
| Offboarding checklist | Knowledge transfer, access revocation |
| Leave policy | PTO, sick leave, parental leave |
| Remote work policy | Expectations, equipment, communication |
| Expense policy | What is covered, approval process |

## Stack Context
- **Notion**: All HR documentation, hiring pipelines, and performance tracking in Notion.
- **GitHub**: For technical roles, use GitHub for code assessments.
- **Resend**: Candidate communication and onboarding emails.

## Quality Gate
- [ ] Organizational assessment completed
- [ ] Hiring plan for next 6-12 months
- [ ] Performance framework defined and communicated
- [ ] Culture documented (values, norms, processes)
- [ ] Key people processes documented

## Anti-Patterns
- **No documentation** — processes that exist only in the founder's head do not scale.
- **Hiring without plan** — reactive hiring (we need someone NOW) leads to bad hires.
- **No performance feedback** — annual reviews only are too infrequent. Monthly 1:1s minimum.
- **Culture by accident** — undocumented culture becomes whatever the loudest person says it is.
