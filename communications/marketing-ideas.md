# Marketing Ideas

## When to Use
- Quarterly marketing planning
- Client asks "what should we do to grow?"
- Existing marketing channels are plateauing
- Need fresh campaign concepts
- Brainstorming session for a new initiative

## Workflow

### Step 1: Audit Current Channels
Before generating new ideas, understand what exists:
| Channel | Active? | Performance | Effort |
|---------|---------|------------|--------|
| Organic search | Yes/No | Traffic/conversions | |
| Email | Yes/No | Open rate/CTR | |
| Social (LinkedIn) | Yes/No | Engagement | |
| Paid ads | Yes/No | ROAS | |
| Referrals | Yes/No | Referral signups | |
| Partnerships | Yes/No | Partner-driven leads | |
| Community | Yes/No | Engagement/leads | |

### Step 2: Generate Ideas (Ideation Frameworks)

**Channel x Format Matrix:**
| | Blog | Email | Social | Tool | Video | Event |
|---|---|---|---|---|---|---|
| SEO | | | | | | |
| Paid | | | | | | |
| Community | | | | | | |
| Partnership | | | | | | |

Fill each cell with a specific idea. This forces creativity across channels and formats.

**Customer Journey Ideas:**
For each stage, brainstorm 3 ideas:
- **Awareness**: How do people discover us?
- **Consideration**: How do we build trust?
- **Decision**: How do we drive action?
- **Retention**: How do we keep them?
- **Referral**: How do we get them to share?

**Competitor Gap Ideas:**
- What are competitors NOT doing?
- What content/channels are they ignoring?
- Where is their messaging weak?

### Step 3: Evaluate Ideas
Score each idea on:
| Idea | Impact (1-5) | Effort (1-5) | Confidence (1-5) | ICE Score |
|------|-------------|-------------|-------------------|-----------|
| | | | | I x C / E |

### Step 4: Select Top 3-5
Pick the highest-scoring ideas. For each:
- Define the specific deliverable
- Set a timeline
- Assign an owner
- Define a success metric

### Step 5: Document and Execute
Create a "Marketing Ideas — [Quarter]" page in Notion with:
- All generated ideas (for future reference)
- Selected ideas with plans
- Timeline and owners
- Success metrics

## Stack Context
- **Notion**: Idea bank and quarterly plans in Notion.
- **Vercel Analytics**: Measure impact of implemented ideas.
- **Next.js**: Many ideas will require website changes (new pages, CTAs, tools).
- **Resend**: Email-based ideas use Resend for delivery.

## Quality Gate
- [ ] Current channels audited before generating new ideas
- [ ] At least 15 ideas generated across multiple frameworks
- [ ] Ideas scored and ranked
- [ ] Top 3-5 selected with specific plans
- [ ] Each selected idea has a timeline, owner, and metric
- [ ] Documented in Notion

## Anti-Patterns
- **Shiny object syndrome** — chasing every new tactic without finishing previous initiatives.
- **Ideas without evaluation** — brainstorming without scoring leads to executing the loudest idea, not the best.
- **Too many ideas at once** — focus beats breadth. 3 ideas executed well beat 10 ideas half-done.
- **No measurement** — every marketing initiative must have a success metric.
- **Copying competitors** — competitor ideas are a starting point, not a strategy.
